## Visual Design

### Appearance
- **Input:** 44px width, 80px height, `1px solid --color-slate-300`, `--radius-md`
- **Padding:** 4px (inner color swatch fills remaining space)
- **Cursor:** pointer
- **Value:** hex string `#RRGGBB`

### States
| State | Border |
|-------|--------|
| Default | `--color-slate-300` |
| Focus | `--color-primary-500` + standard focus ring |

### Usage
For metric type color selection. Native OS picker dialog on click. Do not style beyond recognition.
