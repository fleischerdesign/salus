# Typography

> Font family: **Manrope** (exclusive). Monospace: JetBrains Mono.

## Type Scale

| Token | Weight | Size | Line | Letter | Usage |
|-------|--------|------|------|--------|-------|
| `display` | 800 | 48px | 56px | -0.02em | Hero values |
| `headline-xl` | 700 | 36px | 44px | -0.02em | Page titles (h1) |
| `headline-lg` | 700 | 28px | 36px | -0.01em | Section headers |
| `headline-lg-mobile` | 700 | 24px | 32px | — | Mobile h1 |
| `headline-md` | 600 | 20px | 28px | — | Card titles |
| `body-lg` | 400 | 18px | 28px | — | Long-form text |
| `body-md` | 400 | 16px | 24px | — | Body text |
| `body-sm` | 400 | 14px | 20px | — | Secondary body |
| `label-md` | 600 | 13px | 18px | 0.05em | Nav links |
| `label-sm` | 500 | 12px | 16px | 0.05em | Table headers |
| `caption` | 400 | 11px | 16px | — | Timestamps |
| `code-sm` | 400 | 13px | 20px | — | Inline code |
| `title-sm` | 600 | 14px | 20px | — | Stat labels |

## Utility Classes

```css
.text-xs  { font: var(--font-caption); }
.text-sm  { font: var(--font-body-sm); }
.text-md  { font: var(--font-body-md); }
.text-lg  { font: var(--font-body-lg); }
.font-medium    { font-weight: 500; }
.font-semibold  { font-weight: 600; }
.font-bold      { font-weight: 700; }
.font-extrabold { font-weight: 800; }
```
